# PyTorch: Invertible Neural Network
In this repo we implement an invertible neural network using PyTorch.

## Dependencies

 1. numpy
 2. torch
    *  torch.autograd
    *  torch.optim
    *  torch.nn
 3. [FrEIA][freia]
    *  FrEIA.framework
    *  FrEIA.modules

[freia]: <https://github.com/VLL-HD/FrEIA>
